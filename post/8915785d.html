剑指 Offer 13. 机器人的运动范围

题目

难度：中等

地上有一个m行n列的方格，从坐标 [0,0] 到坐标 [m-1,n-1] 。一个机器人从坐标 [0, 0] 的格子开始移动，它每次可以向左、右、上、下移动一格（不能移动到方格外），也不能进入行坐标和列坐标的数位之和大于k的格子。例如，当k为18时，机器人能够进入方格 [35, 37] ，因为3+5+3+7=18。但它不能进入方格 [35, 38]，因为3+5+3+8=19。请问该机器人能够到达多少个格子？

示例 2：

提示：

  • 1 <= n,m <= 100
  • 0 <= k <= 20

题解

本题和上一题的矩阵路径相似，都是遍历搜索可行路径。根据题意，我们可以使用深度优先遍历（DFS） + 剪枝或者广度优先遍历（BFS）方法来解决。

深度优先遍历（DFS）

  • 递归参数：二维向量 visited(记得带引用&，因为递归过程中需要修改)，坐标 i 和 j ，范围 k

  • 终止条件：

    • 返回 0

      1. 坐标 i 和 j 越界
      2. 坐标之和超出范围
      3. 已访问

    • 返回 1 + dfs(visited, i+1, j, k) + dfs(visited, i, j+1, k)

      • 当前坐标未被访问且符合条件

  • 递推工作：

    1. 标记当前坐标已被访问
    2. 递归计算相邻节点

  • 返回值

    • 返回符合条件的格子数 1 + dfs(visited, i+1, j, k) + dfs(visited, i, j+1, k)

  • 时间复杂度： O(MN)

  • 空间复杂度： O(MN)

广度优先遍历（BFS）

  • 初始化：将坐标 (0,0) 加入到队列中
  • 迭代终止条件：队列为空
  • 迭代工作：
    1. 将队首元素出队
    2. 检查该元素是否符合条件：是否越界、是否已访问和数位和是否超范围
      1. 符合条件：标记该元素已访问并记录统计数 count、将该元素的相邻右下元素入队
      2. 不符合条件：continue
  • 返回值：返回统计数 count
  • 时间复杂度： O(MN)
  • 空间复杂度： O(MN)

本题考点

  • 考查对 回溯法 的理解
  • 考查对 深度优先遍历、剪枝法 和 广度优先遍历 的理解
